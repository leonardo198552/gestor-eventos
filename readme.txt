== Descrição ==
Plugin para gerenciar eventos, emitir ingressos com QR Codes e integrar blockchain para segurança e rastreabilidade.

== Instalação ==
1. Faça upload do plugin para a pasta `/wp-content/plugins/`.
2. Ative o plugin no menu 'Plugins' do WordPress.
3. Use os shortcodes [gestor_event_creation] e [gestor_ticket_view].
